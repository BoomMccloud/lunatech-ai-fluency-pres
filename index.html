Should AI do this — and how much?

Delegation is the first decision: hand off the legwork, keep the judgment — and feed it only public news, never confidential data.

Co-pilot

A vague ask gets a vague answer

So Maya spells out what "good" means — in three layers:

What you want back

5 stories, ~300 words, each with a link.

How to make it

Scan 6 sources → drop dupes → rank by relevance.

Style how it should read

Sharp but friendly. Never invent a source. Flag if unsure.

Same model. Far better output. The difference is the description — but the first try is rarely the last.
Delegation → Describe⇄Discern → Diligence

Read the draft like an editor

That three-layer description was just round one. Maya judges what came back:

🔗 Are the links real? 📅 Anything stale? 🎯 Specific to her industry — or generic filler?

↺ She folds each fix back into the skill — "never summarize an opinion piece as news" — then describes again, smarter.

A few rounds in, the skill runs reliable.
Delegation → Describe⇄Discern → Diligence

It ships under your name

The inner loop got it right — now step back out and own it. Diligence is the habit that keeps that true:

Review you're the last eyes

Read it before it ships. The AI doesn't get the final say — you do.

Be transparent where it matters

Disclose AI's role when it counts — a client deliverable, a sensitive call.

Fixable fast keep an off-switch

Spot a mistake and correct or pull it within minutes.

You're accountable — not the AI.
How to use it · stay safe

Four ways AI bites

🌀 Hallucination

Confident, plausible, wrong — a link that 404s.Verify sources first.

🔒 Bad / private data

Don't paste secrets into a tool you don't control.Mind your inputs.

📅 Stale knowledge

No live source → months-old news as today's.Give it live tools.

🚀 Over-automation

Wiring it to send / pay / delete too soon.Start co-pilot. Keep an off-switch.
